# show-captures

Shows full details of each capture.

````
node src\cli\cli.js show-captures [glob]

Options:
  --help          Show help                                                                [boolean]
  --filter-ua     Filter by userAgent [RegExp]                                              [string]
  --name-max-len  Max length allowed for resource name                        [number] [default: 50]

Examples:
  show-captures --filter-ua Firefox  Shows only captures with userAgent including 'Firefox'.
````

## Examples

Show only JSON files matching `F*45*.json` in any subdirectory of `./test`.

`node src\cli\cli.js show-captures --name-max-len 150 test/**/F*45*.json`

# summarise-captures

Summarises the capture details.

````
node src\cli\cli.js summarise-captures [glob]

Options:
  --help       Show help                                                                   [boolean]
  --filter-ua  Filter by userAgent [RegExp]                                                 [string]

Examples:
  summarise-captures --filter-ua Firefox  Shows only captures with userAgent including 'Firefox'.
````

## Examples

Summarise only JSON files matching `F*.json` in any subdirectory of `./test`.

`node src\cli\cli.js summarise-captures --name-max-len 150 test/**/F*.json`
